# 服务员端 — 功能测试文档

> 测试环境：后端 `localhost:8000` + 前端 `localhost:5173`

---

## 测试准备

```bash
# 终端 1: 启动后端
cd backend
python server.py

# 终端 2: 启动前端
cd waiter
npm run dev
```

浏览器打开 `http://localhost:5173`
